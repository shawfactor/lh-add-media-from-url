Surf the web and add content to the wordpress media library via a bookmarklet

== Description ==

This plugin allow you to grab image from remote url and save into your own wordpress media library. By doing so, you never worried if the remote image was removed by its owner. This also save you steps to download the image to local computer and upload again to your own WordPress. There is also a javascript bookmarklet that helps to automate the process if you are surfing the internet and find something you would like to add to the library

== Installation ==

1. Upload the entire `lh-add-media-from-url` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Media->Add from Url to add media via url or install the bookmarklet.
